3D Room Design Web App

Website applications developed for a client and the IT Systems Project class in the Computer Systems Technology course. The purpose of the application was to allow the client to design 3D rooms and cabinets for his custom cabinet company. This was a continued project from a previous year that had a single page website using Neo4j to store data for the 3D rooms and cabinets.

The project was split into three separate projects, the front end website, an administrator portal, and an API for the projects to communicate to the database. My teams main focus was to develop the administrator portal for a more clear and easy to use interface to manage the database. Many changes had to be made to the original front end website to ensure functionality was maintained, and the end result was a system that could be effectively used to design a room, although more work is still to be done.

Development Period: August 2023 - April 2024

Languages / Frameworks: Javascript, React, Next.js, Cypher

Software: Docker, WebStorm

PF1e Character Builder

This was my final project for the Mobile Application Programming class in the Computer Systems Technology course. The purpose of the application was to allow users to create characters under the pathfinder tabletop role-playing game ruleset. The current version of the application has some functionality to create a level 1 character. Users can use the application to set details such as name, age, gender, race, and attributes for a character.

The application makes use of SQLite database to store information on the different choices for race selection. It could also be utilized to store more tables if it were to be expanded when adding classes, feats, and spells. File based storage is used to store text files that contain the created characters information. The saved file can then be loaded later for further viewing and editing later by users.

Development Date: May 2022

Language: Java

Software: Android Studio

Catching Game Prototype

This is a prototype I created while learning the Unity engine through the Junior Programmer pathway from the Unity Learn website.

I was given a project with some basic starting objects and scripts and the object to expand on it to make a fun game. After a bit of brainstorming I came up with apple catching as an idea and started by creating a simple scene with a tree and attached the box to a pill object.

I created controls for the pill to be able to move around the scene with the WASD keys, added a UI interface that could start and restart the game and keep track of the player's score. Overall, the Unity pathway was a fun way to learn Unity and to learn a bit about programming in C#.

Development Date: August 2022

Language: C#

Software: Unity
